# ShowMessages
show messages and star thems.

#### This project is an example that shows the list of messages from Api and database. You can also saved,delete and share messages in the list and items can be expanded and collapsed

#### This project has features such as:
- Implement MVVM architecture
- Implement DI with dagger2
- Using Flow
- Using Retrofit library for request to server
- Using Room library for store data
- Using Compose libs for implement view
- Using Material Design for UI
- Using coil library for load images
